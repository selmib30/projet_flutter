Description détaillée

Classe de seuillage d'une image.

Auteur
Benjamin Albouy Kissi Cette classe permet d'appliquer le double seuillage à une image. Le double seuillage peut s'appliquer préalablement sur une miniature de l'image complète, permettant d'avoir un aperçu plus rapide du résultat final. L'image originale est concervée intacte de façon à pouvoir modifier les valeurs des seuils.

◆ DoThreshold()

void QLAPINThreshold::DoThreshold ( bool  bPreview)

Calcule le double seuillage sur la miniature ou l'image complète.

Auteur
Benjamin Albouy Kissi
Date
13/09/2010
Paramètres
bPreviewSi true, calcule sur la miniature, sinon sur l'image complète.

Applique le double seuillage sur l'image en miniature (plus rapide) si bPreview vaut true, ou sur l'image complète (plus lent) si bPreview vaut false.

Le traitement est parallélisé pour tirer partie des processeurs multi-coeurs à l'aide d'OpenMP.
